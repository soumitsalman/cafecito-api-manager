# Cafecito API & MCP Developer Portal

Welcome to the developer portal for **Project Cafecito**. This repo and portal are the
overarching home for Cafecito API products.

## Product portfolio

- [Beans API & MCP](/howtos/beans-api) — live news and blog intelligence product. [Pricing](/pricing/beans-pricing)
- [Cortado API & MCP](/howtos/cortado-api) — planned future product. [Pricing](/pricing/cortado-pricing)
- [Latte API & MCP](/howtos/latte-api) — planned future product. [Pricing](/pricing/latte-pricing)

## Start in 3 steps

1. Read [Get API Key](/howtos/api-keys) to generate your token.
2. Open your product guide: [Beans](/howtos/beans-api), [Cortado](/howtos/cortado-api), or [Latte](/howtos/latte-api).
3. Explore endpoint details in product references: [Beans API](/api/beans), [Cortado API](/api/cortado), [Latte API](/api/latte).
